# ISEM-540 Project-2 UI
Things that are used here
 - angularjs 1.5
 - typescript
 - webpack
 - mock-backend
 - npm & node
 
How to run...
 - Nodejs 4.x
 - NPM (3.5 or latest)
 - 
 ```sh
$ git clone https://github.com/tsengel10/ISEM-540-2
$ npm install
$ npm start -- --env dev
```
Then open your browser and http://localhost:8090/isem540/#/
 
To see mock-backend: http://localhost:8090/
